# Compression migration coverage

This is the fail-closed migration ledger from the archived `sx-zip` staging
repository into SX stdlib. The source ledger is
`sx-zip/docs/COVERAGE.md`, pinned to miniz 3.1.2 commit
`77d0dce8627735138c51770d1799a1ef48f2117d`. That ledger inventories all 118
upstream public entry points, internal implementation paths, machine variants,
and compile-time configurations.

The stdlib port is not a C API compatibility layer. Every source-ledger area
is classified here as one of:

- **retained** — the behavior remains available through an idiomatic stdlib
  operation;
- **internal** — the behavior is required for deterministic bytes or
  performance but is deliberately not public;
- **superseded** — only a C ABI/status/flag/configuration spelling was removed;
  its useful behavior is retained elsewhere or the spelling has no native SX
  equivalent;
- **pending gate** (none remain as of the final tree) — a row whose fresh evidence broad
  enough for the historical claim has not yet passed in this worktree.

No row may be called complete while it contains a pending gate.
